# Russian Text Simplifier

##  Обзор проекта

**Russian Text Simplifier** — это веб-приложение на Spring Boot, предназначенное для адаптации русскоязычных текстов к определенным уровням владения языком (B1 и B2 по шкале CEFR). Приложение предоставляет интуитивно понятный интерфейс для упрощения текстов, хранит историю в рамках сессии и генерирует детальные Excel-отчеты с оценкой качества с использованием Textometr API.

### Ключевые возможности
-  Адаптация текстов для уровней B1 и B2
-  Несколько шаблонов промптов для различных подходов к упрощению
-  Хранение истории в рамках сессии
-  Генерация Excel-отчетов с метриками качества
-  Автоматическая оценка качества через Textometr API
-  Визуальные диаграммы, показывающие процент успешной адаптации

##  Основные компоненты

### 1. TranslatorController
Главный MVC-контроллер, обрабатывающий все веб-запросы.

**Эндпоинты:**
| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/translator` | Отображает форму для перевода |
| POST | `/translator` | Обрабатывает запросы на адаптацию текста |
| GET | `/translator/download` | Генерирует и скачивает Excel-отчет |

**Обязанности:**
- Отображение веб-интерфейса
- Обработка запросов на перевод
- Запуск генерации отчетов
- Очистка истории сессии после скачивания отчета

### 2. TranslatorService
Сервис, отвечающий за адаптацию текста и интеграцию с LLM.

**Особенности:**
- Хранение нескольких шаблонов промптов
- Динамическая подстановка целевого уровня владения языком
- Управление вызовами внешнего LLM API
- Возврат адаптированного текста

**Шаблоны промптов:**
```java

#  Russian Text Simplifier — Сценарий работы и документация

##  Путь пользователя

### 1. Открытие приложения

Пользователь переходит по адресу: `/translator`
Видит форму с:

* Полем для ввода текста
* Выбором шаблона промпта
* Выбором уровня (B1 или B2)
* Кнопкой **"Перевести"**

### 2. Отправка текста на адаптацию

Пользователь:

* Вводит русскоязычный текст
* Выбирает шаблон
* Выбирает целевой уровень (B1/B2)
* Нажимает **"Перевести"**

### 3. Обработка

* Текст отправляется в `TranslatorService`
* Выбирается соответствующий шаблон промпта
* Вызывается LLM API
* Адаптированный текст возвращается пользователю

### 4. Сохранение в историю

* Результат сохраняется в сессии
* Пользователь может продолжать добавлять тексты

### 5. Генерация отчета

Пользователь нажимает **"Скачать Excel"**

Происходит:

* Извлечение всех записей из истории
* Анализ каждого текста через Textometr API
* Проверка качества адаптации
* Генерация Excel-отчета с диаграммами
* Скачивание отчета

### 6. Очистка сессии

* История автоматически очищается после скачивания
* Приложение готово к новой сессии

##  Детали Excel-отчета

### Лист: Results

| Столбец              | Описание                                |
| -------------------- | --------------------------------------- |
| Исходный текст       | Исходный текст до адаптации             |
| Адаптированный текст | Упрощенная версия, сгенерированная LLM  |
| Целевой уровень      | Выбранный пользователем уровень (B1/B2) |
| Определенный уровень | Уровень, определенный Textometr API     |
| Комментарий уровня   | Детальный анализ от Textometr           |
| Статус               | ✓ Успешно / ✗ Неуспешно                 |

### Критерии успеха

**Успешно:**

* B1 → уровни 2–3 по шкале Textometr
* B2 → уровни 3–4 по шкале Textometr

**Неуспешно:**

* Уровень вне целевого диапазона

### Визуализация диаграммы

Круговая диаграмма, показывающая:

* Процент успешно адаптированных текстов
* Процент неуспешных

* Java 21 или выше

##  Шаблоны промптов

### Уровень B1

* Разбивать предложения (≤ 10–12 слов)
* Простая грамматика
* Без причастий и деепричастий
* Активный залог
* Простая лексика
* Не более одного придаточного

### Уровень B2

* Сохранять структуру текста
* ≤ 2 придаточных
* ≤ 20 слов в предложении
* Упрощение лексики
* Пояснение терминов
* Избегать сложных цепочек падежей

##  Оценка качества

### Соответствие уровней

| CEFR | Textometr | Описание         |
| ---- | --------- | ---------------- |
| A1   | 1         | Начальный        |
| A2   | 2         | Элементарный     |
| B1   | 2–3       | Средний          |
| B2   | 3–4       | Выше среднего    |
| C1   | 4–5       | Продвинутый      |
| C2   | 5–6       | Профессиональный |

### Метрики качества

Сравниваются:

* Целевой уровень
* Фактический уровень

Позволяет измерить:

* Насколько хорошо упрощён текст
* Достигнут ли целевой уровень
* Стабильность результатов
